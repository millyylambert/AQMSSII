## Project Description

This project examines the determinants of presidential approval across U.S. states
using OLS regression. The analysis estimates two models: a baseline with unemployment
and negative news coverage, and an extended model adding party, term, and region controls.

## Files

| File | Description |
|------|-------------|
| `analysis.R` | Loads the approval dataset, runs two OLS regression models, saves a regression table to `tab/reg_table.tex` and a scatter plot to `img/scatter_approval.pdf` |
| `document.tex` | LaTeX document that loads the table and figure produced by R and compiles into a PDF |
| `img/scatter_approval.pdf` | Scatter plot of presidential approval vs. unemployment rate |
| `tab/reg_table.tex` | Regression table generated by `modelsummary()` |

## How to Reproduce

**Step 1 — Run the R script** from inside the `assignment10/` folder:
```r
source("analysis.R")
```

**Step 2 — Compile the LaTeX document** twice (to resolve references):
```bash
pdflatex document.tex
pdflatex document.tex
```

The output `document.pdf` will contain the figure and regression table automatically.

## Dependencies

**R packages:** `tidyverse`, `ggplot2`, `modelsummary`  
**LaTeX packages:** `tabularray`, `booktabs`, `siunitx`, `graphicx`, `geometry`
