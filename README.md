# Chinese BRI Investment and Institutional Quality
## AQMSS Final Paper

## Overview
This repository contains the R script for my quantitative analysis examining 
whether domestic institutional characteristics influence the allocation of 
Chinese BRI investment across recipient countries.

## Data Sources

**1. AidData Global Chinese Development Finance Dataset (v2.0)**
Must be downloaded manually before running the script.
Download from: https://www.aiddata.org/data/aiddatas-global-chinese-development-finance-dataset-version-2-0
Save the CSV to your working directory and update the file path in the script.

**2. V-Dem Dataset**
Loaded automatically in R via the vdemdata package. No manual download needed.

**3. World Bank WDI**
Loaded automatically in R via the WDI package. No manual download needed.

## How to Run
1. Download the AidData CSV and save it to your working directory
2. Update the file path in the script where indicated
3. Run the R script in order from top to bottom
